# Ansible Role: marvel-nccr.gudhi

An ansible role that installs [GUDHI](http://gudhi.gforge.inria.fr/) on Ubuntu.

## Installation

`ansible-galaxy install marvel-nccr.gudhi`

## Role Variables

See `defaults/main.yml`

## Example Playbook

  - hosts: servers
    roles:
    - role: marvel-nccr.gudhi
